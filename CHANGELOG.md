# Changelog

Все значимые изменения в проекте документируются здесь.

## [0.1.0] - 2026-02-16

### Добавлено
- Первый публичный релиз
- Пайплайн гуманизации текста с 6 этапами обработки
- Поддержка русского, украинского и английского языков
- Автоматическое определение языка
- Профили: `chat`, `web`, `seo`, `docs`, `formal`
- Нормализация типографики
- Деканцеляризация текста
- Разнообразие структуры предложений
- Уменьшение повторов и тавтологии
- Инъекция «живости» для разговорных профилей
- Валидация качества результата
- Защита кода, URL, email, markdown, HTML
- CLI-интерфейс
- Метрики анализа «искусственности» текста
- Подробная документация
